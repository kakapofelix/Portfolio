    Project difficulties

    I used Bootstrap 4 for responsiveness, but some elements of the layout were falling apart and I found Bootstrap is not that customizable that I wanted to.

    I was struggling with the grid of the footer and also sticking it to the bottom.

    My solution

    Bootstrap 4 is one way for making the site responsive, but I started using SASS preprocessor and SASS made things a lot easier. With "mixins" for example, I defined a "mixin" for different screen sizes and it worked out so well. For images, I rendered different images width different size for each screen size, so mobile and small screen size do not need to download big files and it loads faster.

    I solved my footer problem with flexbox, it is super customizable and “flex” their sizes to respond to the available space. The sticking bottom problem I solved with negative margin and it works fine.
