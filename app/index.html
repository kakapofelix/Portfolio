    My work

    The Chess Website
    Follow the hungarian chess grandmaster's tournaments, check his analysis and learn the basics from one of the best chess player in the world. This project will be launched by end of April 2020 soon.
    The languages and tools that I used:

    • HTML / CSS
    • JavaScript
    • NPM, Webpack
    • PostCSS
    • Git

    The goal

    Business aims to achive, making training videos and allowing users with account to access them in long-term.
    Detailed chess analysis where the users can follow up some of main the tournaments.
    Follow the events, past and upcoming tournaments more consistently.

    My role

    Meeting the client regularly and understanding the conception. Each of our meeting has an important role to get some feedback from the client and make modification based requirements.
    After research process, I started Wireframing & prototyping with creating the real content and its design. I gave an estimate and time-scale for design.

    Project difficulties

    I used Bootstrap 4 for responsiveness, but some element were not working out nicely, for example the background picture of the homepage on mobile size.
    I was struggling with the grid system of the footer and sticking it to the bottom. The "div" containers did not align horizontally the way I wanted.

    My solution

    Bootstrap 4 is one good way for making the site responsive, but I finally installed an NPM package - "mixins" and I found it more customizable and gave cleaner code. For images, I used different sizes for each screen size, so mobile and small screen size do not need tup download big files.
    I solved my footer problem with flexbox, it is super customizable and “flex” their sizes to respond to the available space. The sticking bottom problem I solved with negative margin and it works fine.
